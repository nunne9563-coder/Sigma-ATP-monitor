# Security Policy: Sigma ATP Monitor

## 🛡️ Sovereign Security Philosophy

As a component of the **SigmAI Initiative**, the Sigma ATP Monitor prioritizes local privacy and air-gapped security. We believe that security is a core pillar of computational sovereignty.

| Version | Supported |
| :--- | :--- |
| 3.x | ✅ |

## Reporting a Vulnerability

If you discover a security vulnerability within this repository, please **do not open a public issue.** Instead, follow the Sovereign Disclosure protocol:

1.  **Contact**: Send a secure message or report via the [nunne9563 GitHub Profile](https://github.com/nunne9563).
2.  **Description**: Provide a clear description of the vulnerability, including steps to reproduce it.
3.  **Response**: We will acknowledge your report and coordinate a fix before any public disclosure.

## Hardening Recommendations

- **Local-Only**: This monitor is designed to run without network access. For maximum security, use it in isolated environments.
- **Dependency Audit**: We maintain minimal external dependencies to reduce the attack surface.
